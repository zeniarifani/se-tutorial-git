Software Engineering Git Practice SEMANGAT

Welcome to the Git practical session. This webpage is part of your hands-on tutorial to understand Git and GitHub workflows.

Your Task

  1. Fork this repository (https://github.com/alfhisa/se-tutorial-git)
  2. Clone the forked repository
    • git clone https://github.com/alfhisa/se-tutorial-git.git
    • cd se-tutrial-git
  3. Create a new branch
    • git checkout -b tutor-git (you may name the branch as you like)
  4. Edit this file (index.html).
    • Edit the file as creatively as possible
    • You can also add more files if you want
  5. Commit and Push your changes
    • git add .
    • git commit -m "Edit the index.html file" (you may change the message)
    • git push origin tutor-git (adjust with your branch name)
  6. Open a pull request on GitHub
    • Visit your forked repo
    • Click "Compare & pull request"
    • Fill out the title and description
    • Submit the pull request
